# Fotos de perfil

Pega aquí los archivos de imagen (`.png`, `.jpg`, `.jpeg`, `.webp`) que quieras
ofrecer como avatar. Se detectan automáticamente al recargar la app — no hace
falta tocar código ni registrarlos en ningún lado.

El nombre de archivo es el identificador que se guarda para cada usuario, así
que evita renombrarlos una vez que la gente ya los haya elegido.
