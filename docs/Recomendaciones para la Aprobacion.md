# Resumen: Cómo hacer que aprueben tu sitio web en Google AdSense

En este video de **Google AdSense**, el presentador Andreas comparte las tres mejores prácticas y recomendaciones clave para aumentar considerablemente las posibilidades de que un sitio web sea aprobado y su estado cambie a "Listo" para mostrar anuncios. https://www.youtube.com/watch?v=lZUG0XGlZZY

---

## 1. Asegurar que el código de anuncio sea correcto y esté completo
Para que el sistema de AdSense pueda verificar tu sitio, debes colocar el código de manera adecuada:
* **Ubicación:** Copia el código de anuncio generado exactamente como aparece en tu página de inicio de AdSense y pégalo en la etiqueta `<head>` o en la parte superior del `<body>` de tu página, sin realizar ninguna modificación [00:00:43].
* **Coincidencia de sitio:** Asegúrate de colocar el código en el mismo sitio web que registraste en tu cuenta de AdSense [00:00:52].
* **Alternativa de verificación:** También puedes verificar la propiedad de tu sitio a través de **Google Search Console**. Si el sistema detecta que el sitio ya está verificado allí, continuará con el proceso de revisión de idoneidad sin problemas [00:00:58].

---

## 2. Garantizar que el sitio web sea accesible (Reachable)
El equipo de AdSense no puede revisar sitios web que estén caídos o no carguen correctamente. Asegúrate de cumplir con lo siguiente:
* **Disponibilidad global:** El sitio web debe estar activo y funcionando correctamente a nivel mundial [00:01:12].
* **Sin protección por contraseña:** Si tu sitio requiere un inicio de sesión o contraseña para acceder, considera desactivar temporalmente esta pantalla de bloqueo para que los revisores y los rastreadores puedan acceder [00:01:25].
* **Archivo robots.txt:** Verifica que no estés bloqueando el rastreador (crawler) de Google en tu archivo `robots.txt` [00:01:39].

---

## 3. Cumplir con las políticas del programa (Policy Compliant)
Es fundamental revisar las políticas del programa y de contenido. El video destaca las **tres infracciones más comunes** que cometen los editores y que debes evitar a toda costa:
* **Contenido de bajo valor (Low value content):** El contenido de tu sitio debe ser original, relevante y aportar valor real a los usuarios [00:01:59].
* **Contenido duplicado o replicado (Replicated content):** Asegúrate de que tu contenido sea único y no se repita en diferentes páginas de tu propio sitio, ni esté copiado de otros sitios web [00:02:13].
* **Navegación del sitio (Site navigation):** Los usuarios deben poder navegar por tu sitio de manera sencilla. Utiliza barras de menú y categoriza el contenido de forma clara [00:02:25].

> **Nota importante:** AdSense puede revisar **todas** las páginas de tu sitio, no solo la URL principal con la que te registraste. Por ello, cada página individual debe cumplir con estas políticas [00:02:38].

---

## Consejos adicionales para acelerar el proceso
* **Página popular:** Se recomienda colocar el código de anuncio en la página más popular o con más tráfico de tu sitio web, ya que esto ayuda a que el sistema de AdSense reconozca el código más rápido y acelere la activación [00:02:49].
* **No elimines el sitio:** Mantén tu sitio dentro de la lista de sitios de tu cuenta de AdSense; eliminarlo durante la revisión podría retrasar significativamente el proceso [00:03:01].
